# TTS-Edge-Azure 简洁高效的 TTS 工具

一个专注于「快速、清晰、易用」的文本转语音 (TTS) 应用，旨在将任意文字内容瞬间合成高质量语音。无论是阅读文章、校对文稿，还是在多任务场景中获取语音提示，它都能为你提供流畅、自然的声音体验。

## 核心功能

- **通用文本输入**：支持直接粘贴一键开始朗读。
    
- **多声线选择**：内置多种发音人，可灵活切换。
    
- **可调参数**：自由设定**语速 (Rate)**、**音量 (Volume)**、部分引擎还可调整**音高 (Pitch)**。
    
- **简洁界面**：直观的网页式操作，零学习成本。
    
- **播放控制**：提供播放、暂停、停止、进度拖拽、倍速切换、下载音频等核心功能。
    
- **跨平台潜力**：基于 Python + Flask 后端和标准前端技术，适配 Windows、macOS、Linux。
    
- **音频保存**：支持将合成结果导出为 MP3 文件，便于后续分享与归档。
    

---

## 技术架构

- **后端**：Python 3.8+ 与 Flask
    
- **TTS 引擎**：`edge-tts`（Edge 浏览器官方语音合成），可扩展 Azure TTS
    
- **前端**：HTML5 + CSS3 + 原生 JavaScript
    
- **依赖管理**：`venv` + `requirements.txt`

## 安装指南

1. **准备环境**
    
    - 安装 Python 3.8 或更高版本，并确保已添加到 `PATH`。
        
2. **下载仓库**
    
    ```bash
    git clone https://github.com/你的用户名/TTS-Edge-Azure.git
    cd TTS-Edge-Azure
    ```
    
3. **一键启动（Windows）**  
    双击根目录下的 `一键启动.bat`，脚本将自动：
    
    - 创建/激活虚拟环境
        
    - 升级 pip、安装依赖
        
    - 启动 Flask 服务
        
4. **手动方式（跨平台）**
    
    ```bash
    python -m venv venv
    # Windows
    venv\Scripts\activate
    # macOS/Linux
    source venv/bin/activate
    pip install -r backend/requirements.txt
    python backend/app.py
    ```

## 快速使用

1. **访问界面**：启动后打开浏览器，访问 `http://127.0.0.1:5000`。
    
2. **输入文本**：粘贴或上传文本。
    
3. **选择发音人**：从下拉菜单中选取喜欢的声音。
    
4. **调整参数**：拖动滑块设置语速、音量、音高。
    
5. **生成语音**：点击「播放」按钮，实时合成并朗读。
    
6. **下载音频**：在播放完成后，可点击「下载」保存 MP3 文件。

## 语音引擎与参数

- **edge-tts**：基于 Microsoft Edge 的本地/在线合成，支持多种语言和声线。
    
- **azure-tts**（可选）：接入 Azure Cognitive Services，需自行申请 Key。
    
- **参数说明**：
    
    - **Rate**：语速，范围通常在 0.5–2.0。
        
    - **Volume**：音量，0–100。
        
    - **Pitch**：音高，±20%。

## 常见问题

1. **安装依赖过慢或失败？**
    
    - 建议切换到国内镜像源（如清华、阿里）。
        
    - 确认网络通畅并关闭代理/杀毒干扰。
        
2. **界面打开后按钮无响应？**
    
    - 检查浏览器控制台是否有脚本错误，确保 `common.js`、`main.js` 等文件按顺序加载。
        
    - 确保后端 Flask 服务正常运行，无端口冲突。
        
3. **语音合成卡顿或断流？**
    
    - 长文本可分段朗读，或适当调慢语速。
        
    - 检查网络质量（若使用在线引擎）。
        

---

## 注意事项

- 使用在线 TTS 服务时请保持稳定网络。
    
- 极长文本处理会占用更多内存和时间，建议拆分。
    
- 确保本地音频设备正常工作。
